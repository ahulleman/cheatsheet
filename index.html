<HTML> BASICS

Name Tag Description / Function
Head Element <head>...</head> The head element contains the information about your html like meta data, page title etc.
Header <h1-6>...</h1-6> h1 is your largest header and h6 your smallest. It is considered best practise not to mix different size headers across one page.
Title <title>...</title> The title names your page and will show in your current tab name (top of your browser if supported)
Body Element <body>...</body> Anything written in the body element will show in the main window.
Paragraph <p>...</p> A paragraph is used to split up certain sections/chapters of text. Combined with a class or ID they can make styling a lot easier.
Line Break <br> Adds a "linebreak". Bit like pressing the ENTER key in a word processor.

<Tables> & <Lists>

Name Tag Description / Function
Table <table>...</table> The table element is used to create a table. The contents are written out row by row.
Table Header <th>...</th> The table header element represents the heading of either a column or a row.
Table Row <tr>...</tr> The table row element indicates the start of every row and is usually followed by the "table data" element.
Table Data <td>...</td> Each cell of a table is represented using a table data element.
Table Footer <tfoot>...</tfoot> The footer element goes on the lower part of your table and is usually applied for styling purposes.
Colspan/Rowspan <td colspan="2">...</td> Stretches your entries over several columns (Colspan) or rows (Rowspan). These two elements can be used with the "td" and the "tr" elements.
